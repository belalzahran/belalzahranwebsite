Kumon Student Tracker System

Technologies: RaspberryPi, Python, SQL, JSON, Flask, Twilio API, JavaScript, HTML, CSS

Project Description

During the summer of 2023, I undertook a freelance development project to create a Student Tracker system for a Kumon Tutoring Center. This system was designed to streamline their student data and attendance management process. My role in this project included everything from conceptualization to deployment. This system is still being used untill today to manage a center upwards of 150 students!

Watch the video below to see how my Student Tracker Works!


Technologies Used

The final product utilized a variety of technologies and consisted of a Python GUI application programmed on a Raspberry Pi with a tablet interface for easy on-site interaction, a MySQL database also hosted on the Raspberry Pi, and a custom developed Web Interface as means for accessing stored for instructor use. The Web Interface was built using the Flask Framework, Python, HTML, CSS, and Javascript.
